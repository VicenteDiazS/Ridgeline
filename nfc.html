iPhone Workflow

Use URL Tags For Safari

iPhone Safari can open an NFC tag that contains a website URL, but it cannot directly read or write tags from inside this web page.

1 Select the truck location

Pick the fuse box, engine view, service record, or truck area that the tag should open.

2 Copy the URL

Use the URL field from this page. For permanent tags, use the real site address, not localhost.

3 Write it as URL / URI

Paste that URL into an iPhone NFC writer app as a website URL record, then write the tag.

4 Scan from iPhone

After the tag is written, scan it with the iPhone unlocked and tap the Safari banner.

On this device, the page is checking whether direct NFC access is available.
